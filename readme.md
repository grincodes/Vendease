
## Steps To Run Application
- npm i
- npm run run:migrations (To setup tables)
- npm run start:dev



## Run Migration
- Create Migration File: `npm run typeorm:create-migration --name="Create-User-Table"`
- Rum  Migration: `npm run run:migrations"`
